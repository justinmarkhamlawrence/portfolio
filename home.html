Trip Hopper is an app that allows the user to plan a trip by locating nearby points of interest based on an initial starting point. For example, start by adding a restaurant. From there the user can choose to find a nearby coffee shop, or anything else! The trips are saved in a database and can be revisited later! Key technologies include React, Redux, React-Router, Node, Express, Passport and MongoDB. Data is consumed from the Yelp Api as well as GoogleMaps and Google OAuth.

German X is an app that teaches the user german words using a unique algorithm that allows for spaced repetition. The users progress is saved to a database, and determines which words the user needs to work on most. Key technologies include React, Redux, Node, Express, Passport and MongoDB.
