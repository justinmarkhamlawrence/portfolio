Projects

Trip Hopper is an app that allows the user to plan a trip by locating nearby points of interest based on an initial starting point. For example, start by adding a restaurant. From there you could grab a cup of coffee, before taking in a movie! Each point of interest will be based off of the previous one, allowing the user to make a trail of events.The trips are saved in a database and can be revisited later! Key technologies include React, Redux, React-Router, Node, Express, Passport and MongoDB. Data is consumed from the Yelp Api as well as GoogleMaps and Google OAuth.

German X is an app that teaches the user german words using a unique algorithm that allows for spaced repetition. The users' progress is saved to a database, and determines which words the user needs to work on most. Key technologies include React, Redux, Node, Express, Passport and MongoDB.

Dependencies is a Node.js utility that allows the user to extract the dependency modules of any Javascript file that is supplied as the command line argument. Once the source code has been cloned locally, the user will navigate to the location of the directory in the terminal. With Node installed, the user will then enter "node dependencies path/to/file", with path/to/file representing the path to the file in question. Dependencies will then return a json object containing the variables that are used in the file as the key, and the required Node modules or file paths as the values.
