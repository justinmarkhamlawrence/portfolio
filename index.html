Trip Hopper is an app that allows the user to plan a trip by locating nearby points of interest based on an initial starting point. Key technologies include React, Redux, React-Router, Node, Express, Passport and MongoDB. Data is consumed from the Yelp Api as well as GoogleMaps and Google OAuth.
